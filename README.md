# 📊 Java Data Processing Application 

##  Contexte
Ce projet a été réalisé dans le cadre de l’unité d’apprentissage **UA3 – Programmation Avancée**.  
L’objectif est de mettre en pratique les concepts de programmation orientée objet en Java.

---

##  Objectif du projet
Développer une application Java permettant de gérer des étudiants, leurs notes et le calcul de leur moyenne.

---

##  Fonctionnalités principales
- Création d’une classe `Etudiant`
- Gestion des informations d’un étudiant (nom, prénom, notes)
- Calcul automatique de la moyenne
- Lecture des données à partir d’un fichier CSV

---

## Technologies utilisées
- Java
- Programmation orientée objet (POO)
- Fichiers CSV

---

##  Structure du projet
- `Etudiant.java` : représente un étudiant avec ses informations et sa moyenne  
- `notes_etudiants.csv` : fichier contenant les notes des étudiants  
- Dossier `UA3` : code source du projet

---

## ✅ Résultats
Le programme permet de manipuler les données des étudiants et d’obtenir leur moyenne de manière structurée et automatique.

---

##  Ce que j’ai appris
- Appliquer les principes de la programmation orientée objet
- Structurer un projet Java
- Travailler avec des fichiers CSV
- Améliorer la lisibilité et l’organisation du code
